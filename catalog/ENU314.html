    AWS IoT SiteWise Monitor allows you to build portals and dashboards for visualizing asset measurements on AWS IoT SiteWise. AWS IoT Application Kit is a development library for building industrial IoT web-based applications; it helps retrieve data from AWS IoT SiteWise and AWS IoT TwinMaker and provides frontend components and utilities. In this workshop, build a React web application with AWS Amplify and, using components of AWS IoT Application Kit, connect to AWS IoT SiteWise to incorporate live visualizations of telemetry data into the application. You must bring your laptop to participate.
