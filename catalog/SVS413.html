    Gain comprehensive visibility into modern, distributed applications on Amazon ECS with AWS managed open source observability services. Empower effective troubleshooting and performance optimization by leveraging AWS Distro for OpenTelemetry (ADOT), Amazon Managed Service for Prometheus, and Amazon Managed Grafana to gain comprehensive insights into your Amazon ECS environments. This chalk talk demonstrates how to collect metrics, traces, and logs from diverse workloads using ADOT’s service-based collectors seamlessly integrated with Amazon ECS. Learn how to build a unified observability solution for Amazon ECS applications, which enables proactive monitoring, troubleshooting, and optimization even as applications grow more complex.
