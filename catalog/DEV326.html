    Infrastructure as Code (IaC) with AWS CloudFormation provides a productive and safe way to manage infrastructure. The new Pkl declarative language and the CloudFormation Pkl library can bring further productivity gains through better type-safety, reusability with functions and data types, and great IDE support. When used with the Rain CLI, infrastructure can be defined with Pkl and easily tested/deployed. This chalk talk includes an introduction to writing CloudFormation templates with Pkl and using Rain to deploy them.
