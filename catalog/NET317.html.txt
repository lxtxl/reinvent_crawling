AWS 애플리케이션 네트워킹 서비스는 개발자 및 네트워크 관리자가 확장 가능한 애플리케이션을 구축 할 수 있도록 완전히 관리되는 네트워킹 솔루션을 제공합니다.이 세션에서는 ELB (Elastic Load Balancing), Amazon API Gateway, AWS Privatelink 및 Amazon VPC 격자와 같은 서비스를 가장 잘 사용하는 방법을 배우므로 안전한 응용 프로그램 연결 및 모니터링이 가능합니다.보안을 향상시키고 가용성을 극대화하며 연결성을 향상시키고 팀의 워크로드를 줄 이도록 설계된이 서비스 포트폴리오의 최신 기능에 대한 정보를 유지하십시오.응용 프로그램 개발 및 배포 프로세스를 자동으로 간소화하기위한 모범 사례를 배우십시오.
