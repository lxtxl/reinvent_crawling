    Organizations use Amazon EBS snapshots to securely protect data on Amazon EBS volumes. Join this builders’ session to learn how to design and build automated application-consistent Amazon EBS snapshot workflows using Amazon Data Lifecycle Manager. Build snapshot automation into your existing data protection scripts, and explore how to design and code new snapshot workflows. This session also covers topics such as AWS account security, securing your snapshots from public access, secure sharing, and the Amazon EBS snapshot encryption architecture. Learn how to protect your data from accidental deletions, increase compliance, and safeguard against ransomware. You must bring your laptop to participate.
