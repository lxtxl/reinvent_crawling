AWS 서비스가 다양한 규정 및 표준 준수를 단순화하고 자동화하는 데 어떻게 도움이되는지 알아보십시오.이 세션은 규정 준수를 정의하고 자동화의 이점을 논의함으로써 시작됩니다.그런 다음 AWS Artifact 및 AWS Audit Manager를 사용하여 준수 데이터 모음을 자동화하는 방법과 AWS Config 및 AWS 보안 허브로 자동 보안 및 규정 준수 검사를 만드는 방법에 대해 알아보십시오.마지막으로 AWS Systems Manager 자동화 문서를 사용하여 자동 치료 워크 플로 설정을 설정하는 방법에 대해 알아보십시오.세션이 끝날 무렵, 클라우드 보안 엔지니어와 클라우드 아키텍트는 AWS에서 포괄적이고 자동화 된 준수 워크 플로를 설정하는 방법을 알게 될 것입니다.
