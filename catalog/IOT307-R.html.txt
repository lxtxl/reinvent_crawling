이 빌더 세션에서는 오프라인 LLM 및 웹 포털을 AWS IoT Greengrass 구성 요소로 배포하는 방법에 대해 알아보십시오.지역 지식 기반을 구축하기 위해 개인 소스 또는 Amazon S3에서 상황에 맞는 데이터를 동기화하는 방법을 알아보십시오.보다 관련성 있고 정확하며 맞춤형 응답을 위해 검색 증강 생성 (RAG)을 탐색하고 복잡한 에지 작업을 위해 멀티 모달 로컬 에이전트를 통합하는 방법을 배우십시오.이 접근법은 Edge에서 LLM 전원을 잠금 해제하여 상황에 맞는 지식을 Edge Computing 및 Multimodal Agent 기능과 결합하여 데이터 프라이버시, 낮은 대기 시간 및 맞춤형 경험을 보장합니다.참여하려면 노트북을 가져와야합니다.
