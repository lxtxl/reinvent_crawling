SQL Server 비용 최적화는 AWS 사용자에게 최우선 과제입니다.데이터베이스 워크로드는 일반적으로 높은 메모리와 IOP에 의존하고 VCPU에 적합하기 때문입니다.이 세션에서는 Amazon EC2에서 최적화 CPU 기능을 사용하여 가격 성능을 높이는 동시에 Amazon EC2에서 SQL Server를 실행하는 비용을 줄이는 방법을 배우십시오.이 세션에서는 HammerDB 시뮬레이션으로 엄격한 부하 테스트를 포함하여 광범위한 테스트를 통해 얻은 통찰력을 공유합니다.성능을 희생하지 않고 절감 효과를 극대화하는 데 사용할 수있는 모범 사례를 배우십시오.
