    Migrating VMware-based workloads to AWS requires careful technical decisions. This chalk talk provides a roadmap for a seamless transition, covering crucial aspects like developing a data-driven business case, selecting appropriate discovery tools and assessments, establishing AWS foundations, and choosing the right migration and modernization tools. Attendees gain familiarity with most commonly used AWS migration services such as AWS Application Discovery Service, AWS Application Migration Service, and equivalent AWS Partner tools. Gain insights into key technical decisions involved in each phase, allowing you to navigate the migration journey with confidence and minimize disruptions while leveraging the power of the AWS Cloud.
