이 번개 대화는 한국어로 전달됩니다.한국의 주요 식당 예약 서비스 인 Catchtable은 앙상블 방법을 사용하여 AWS에 대한 대기 시간 예측 기능을 시작하여 사용자 경험을 향상 시켰습니다.이 Lightning Talk에서 Catchtable이 Amazon Sagemaker 캔버스를 활용하여 데이터 준비, 모델 생성 및 배포를 생산에 원활하게 통합하는 데이터 파이프 라인을 구축하는 방법을 배우십시오.노 코드 접근 방식을 사용하여 머신 러닝 (ML) 모델 개발 여정을 발견하고 특정 요구 사항을 충족하고 자연 언어 기반 쿼리 (텍스트 투 코드)를 사용하여 코드를 사용자 정의하는 방법을 배우기 위해 필요한 코딩 시간을 줄입니다.전처리.
