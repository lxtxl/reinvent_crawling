    Amazon Virtual Private Cloud (Amazon VPC)
    Monday, Dec 02
    11:00 a.m. Monday, Dec 02
    Embark on a journey to explore best practices for IP addressing on AWS. In this chalk talk, learn how to build a VPC IP addressing plan—including deciding between global unicast addresses (GUAs) and unique local addresses (ULAs) for IPv6 based on their effects on security, connectivity, and network architecture—to ensure scalability. This talk also covers strategies for efficiently using public IPv4 addresses and reducing cost. You walk away with an understanding of how to use Amazon VPC IP Address Manager (IPAM) capabilities to build an IP addressing plan that is extensible as you grow your applications on AWS.
