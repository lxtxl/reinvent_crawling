    Have you ever wanted to gain insights from Jira ticket data without having to read through them all yourself? Have you thought about using AI to gain insights? This dev chat demonstrates how you can connect to and query the Atlassian Jira API to pull ticket data using pagination and more advanced coding practices to create a repository of information. This repository can then be cleaned using data science libraries. Then, see how the cleaned data can be fed to Amazon Bedrock and queried for insights to help improve customer FAQ documentation and increase customer satisfaction.
