이 세션에서는 자체 호스팅 및 클라우드 환경 모두에 대해 Github Enterprise를 위해 계획된 흥미로운 새로운 기능을 발견했습니다.Github가 엔드 투 엔드 데브 팝 플랫폼에서 생산성을 가속화 할 수있는 방법을 살펴보십시오.Github의 엔터프라이즈 제품을 시작하는 방법에 대한 통찰력을 얻고 조직 내에서이를 채택하는 방법을 배우십시오.데모를보고 EMU (Enterprise Managed Ouser)와 로그인하고 프로젝트를 탐색하고 GitHub 코드 스페이스 및 GitHub Copilot과 함께 코드를 기여하고 GitHub 작업 및 코드 스캔으로 리포지토리 규칙을 시행하는 방법을 배웁니다.이 프레젠테이션은 AWS 파트너 인 Github에 의해 제공됩니다.
