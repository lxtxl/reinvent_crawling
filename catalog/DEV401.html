    Security invariants are critical guardrails in cloud environments, but implementing them at scale is challenging. Deploying them into existing environments without breaking things is even harder. This session defines invariants and demonstrates how to build and deploy them at scale. Learn how to use account and org-level settings like BPA and IMDSv2 and empower automated remediation before we get into the details of IAM and SCP creation. This session is filled with practical examples, including automated testing and AI generation support.
