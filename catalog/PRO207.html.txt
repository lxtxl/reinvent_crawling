의료 데이터의 80% 이상이 구조화되지 않았으므로 의료 데이터에 전통적인 기계 학습 (ML) 접근법을 적용하는 것은 중요한 과제입니다.이 Lightning Talk에서 Amazon Bedrock, Amazon Textract 및 기타 AWS 기본 서비스를 구현하여 비정형 의료 문서 데이터를 효율적으로 처리하고 ML 모델을 교육하여 임상 의사 결정에 정보를 제공하는 방법을 배우십시오.대규모 데이터 엔지니어링, 대규모 언어 모델 주식 문서 추출 및 이해, ML 교육을 결합한 솔루션을 구축하는 방법을 알아보십시오. 문서를 처리하고 임상 주제 전문가가 알리는 구조화 된 데이터 모델을 생성하고 ML 모델 성능을 평가하십시오.
