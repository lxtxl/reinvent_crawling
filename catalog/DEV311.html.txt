개발자를위한이 Chalk Talk에서는 AI 기반 보안 서비스를 일일 소프트웨어 개발 라이프 사이클 (SDLC)에 원활하게 통합하여 코드 보안을 향상시키는 방법을 배우십시오.개발, 빌드, 배포 및 애플리케이션 런타임 단계에서 애플리케이션을 구축하기위한 모범 사례에서 AI 기반 보안 서비스를 사용하여 Shift-Left Security Approach를 탐색하십시오.보안 결과를 정상화하고 통합하여 응용 프로그램 보안에 대한 통일 된 가시성을 얻는 방법을 확인하십시오.마지막으로, Code Security Management UI를 통해 생성 AI를 활용하여 보안 결과를 분석하고 간소화하는 방법을 알아보십시오.
