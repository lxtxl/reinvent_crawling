    This bootcamp is exclusively for AWS Partners. It focuses on the rehost (lift-and-shift) migration strategy, utilizing AWS Application Migration Service to bring workloads to the AWS Cloud with minimal architectural changes. Designed for technical architects, migration specialists, system administrators, and project managers involved in migration projects, this bootcamp teaches you how to execute migrations using Application Migration Service. Prerequisites required for this bootcamp include access to an AWS account, familiarity with AWS Management Console and Application Migration Service, and a basic understanding of Amazon VPC, AWS Identity and Access Management (IAM), and AWS Elastic Disaster Recovery. You must bring your laptop to participate.
