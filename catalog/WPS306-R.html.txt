Amazon Q는 AWS 고객이 조직의 생성 AI의 힘을 빠르게 잠금 해제 할 수 있도록 도와줍니다.AWS 공공 부문 고객은이 값을 쉽게 잠금 해제 할 수 있지만 규제 된 워크로드를 운영하는 고객은 보안 및 관리 가능성을 높이기 위해 추가 가드 레일과 제어가 필요합니다.이 Chalk Talk에서 Amazon Q가 신뢰할 수있는 Secure Enclaves 및 AWS 구성과 결합하여 규제 된 워크로드에 대한 데이터를 안전하게 배포하고 관리하는 동시에 안전하고 순응 한 AWS 환경 내에서 생성 AI의 변형 가능성을 가능하게하는 방법을 알아보십시오.
