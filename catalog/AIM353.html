    This code talk dives deep into the evaluation of agentic applications built with large language models (LLMs), leveraging Amazon Bedrock and open source frameworks. Attendees will gain insights into concepts like Retrieval Augmented Generation (RAG), pipelines, agent evaluation, and LangSmith—a unified platform for debugging, testing, evaluating, and monitoring LLM applications. The talk explores techniques for assessing performance metrics, such as quality, cost, and latency, across the development lifecycle. Additionally, it highlights the role of LLM judges and agent metrics in evaluating retrieval, request quality, and overall application performance, enabling attendees to build robust and reliable agentic solutions.
