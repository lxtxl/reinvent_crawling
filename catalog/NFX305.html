    AWS Database Migration Service (AWS DMS)
    Amazon Aurora for PostgreSQL
    Wednesday, Dec 04
    2:00 p.m. Wednesday, Dec 04
    At Netflix, we wanted to migrate from Amazon RDS for PostgreSQL to Amazon Aurora PostgreSQL to address limitations in scalability, availability, and crash recovery of single-node databases. To support this migration, we developed a self-service system that allows customers to effortlessly migrate RDS instances to Aurora clusters. This chalk talk outlines our automated migration tool, which we built internally using AWS Database Migration Service (DMS), and explores the challenges faced in migrating Netflix’s diverse fleet of customer-managed RDS instances.
