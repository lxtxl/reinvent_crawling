    Learn how applying AWS leadership principles and leveraging community helped transform an airline notification service migration to AWS. The project languished for two years without a development environment, caught between missed deadlines and budget overruns. Strategic planning and execution led to a full production release in less than nine months. By using Amazon ECS, Amazon SQS, Amazon SNS, Amazon SES, Amazon S3, AWS Lambda and Terraform (IaC), the team overcame significant challenges. This talk offers key lessons in IT project management, Dev and Ops collaboration, and AWS architecture solutions for achieving robust, scalable architectures.
