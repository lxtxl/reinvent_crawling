이 실습 워크샵에서는 AWS의 Apache Icesberg 기반 데이터 호수에 신뢰할 수 있고 확장 가능한 스트리밍 섭취 파이프 라인을 구축하여 진화하는 데이터 요구 사항에 적응할 수있는 고도로 사용 가능한 실시간 데이터 플랫폼을 구축하는 방법을 배웁니다.일련의 가이드 연습을 통해 AWS 스트리밍 서비스를 사용하여 Amazon S3의 빙산 테이블에 데이터를 수집하고 Amazon EMR을 사용하여 데이터를 처리하는 방법을 배우십시오.스키마 변경 및 데이터 오류를 우아하게 처리하기 위해 원자 쓰기, 스키마 진화 및 메타 데이터 관리를위한 오류 처리를 포함한 Iceberg의 주요 기능을 구현합니다.참여하려면 노트북을 가져와야합니다.
