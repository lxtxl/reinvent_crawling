    This chalk talk outlines how automotive companies can migrate cloud-native development platforms from the AWS partition to the AWS China partition, referencing the successful migration of BMW’s autonomous driving (AD) platform as an example. This talk outlines the technical and organizational challenges that can be expected, as well as the tools to simplify the process. Finally, learn how BMW keeps both the AWS AD and AWS China AD platforms in sync, enabling a cost-effective, long-term operation.
