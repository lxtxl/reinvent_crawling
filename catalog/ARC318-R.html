    This is a whiteboard-lead session that shows how Kuiper's new Low-Earth Orbit (LEO) satellite network works with AWS services and the types of new use cases that are available. Kuiper will provide high-bandwidth, low-latency connectivity almost anywhere on Earth, creating many new use cases when combined with AWS services such as private networking and computing at the edge. This session covers Kuiper basics such as the constellation, design, and user terminals as well as more advanced architecture patterns such as the integration of private endpoint connectivity directly into AWS data lakes or Amazon VPC connectivity.
