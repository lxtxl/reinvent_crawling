AI는 정보를 발견하고 소비하는 방식에서 운전과 쇼핑 방식에 이르기까지 우리가하는 모든 일의 원동력이되었습니다.그러나 어떤 힘이 AI?대답은 API입니다.이 세션에서는 Postman CTO Ankit Sobti 및 업계 리더가 협업이 API 성공의 열쇠, 특히 AI 시대에 어떻게 공유되는지 공유합니다.API 전문가이든 방금 시작하든 API 제품의 성공을 극대화하고 협업 마찰을 줄이며 개발자, 파트너 및 고객에게 세계적 수준의 경험을 제공하는 데 필요한 정보를 얻으십시오.이 프레젠테이션은 AWS 파트너 인 Postman이 제공합니다.
