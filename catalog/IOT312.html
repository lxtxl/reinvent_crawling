In this chalk talk, learn foundational concepts for connecting smart home devices to the cloud and building conversational generative AI systems to interact with the physical world. Using large language models, explore how to integrate sensor data, automate smart home devices, and orchestrate applications to generate actions and recommendations based on context from sensors or cloud data sources.
