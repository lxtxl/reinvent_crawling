조직은 데이터를 사용하여 재무 계획에서 생성 AI에 이르기까지 모든 것을 권장합니다.
데이터 중심이 되려면 조직은 빠르게 진화하는 현대적인 데이터 환경을 따라 잡기 위해 확장 가능하고 효율적인 데이터 파이프 라인을 구축해야합니다.
이 세션에서는 조직이 Apache Spark와 같은 오픈 소스 도구 및 EKS의 Amazon EMR과 같은 AWS 관리 서비스와 같은 오픈 소스 도구를 사용하여 
Petabyte-Scale 데이터 처리 파이프 라인을 구축하는 방법을 살펴보십시오.스토리지, 네트워킹 및 컴퓨팅을위한 주요 아키텍처 결정을 배우고 조직의 목표를 
달성하기 위해 비용 또는 성능을 위해 데이터 파이프 라인을 최적화 할 수 있습니다.
