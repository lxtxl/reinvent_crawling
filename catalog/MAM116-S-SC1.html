    For financial services organizations, mainframe modernization provides significant value focused on improved agility, enhanced customer experiences, and cost savings. To accelerate this process, T. Rowe Price decided to exit the mainframe and move their remaining core applications to AWS, working with Accenture to use code refactoring and generative AI capabilities. Integrating generative AI tools helps automate legacy code analysis and generates comprehensive technical and business artifacts. These artifacts are not just useful for IT and system architects but provide value to business users with accurate insights into legacy application processes that enhance decision-making, improve collaboration, and mitigate risks. This presentation is brought to you by Accenture, an AWS Partner.
