Kubernetes 용 AWS 컨트롤러 (ACK)는 Kubernetes API의 AWS 서비스를 직접 관리 할 수있는 오픈 소스 프로젝트입니다.이 Lightning Talk는 먼저 친숙한 Kubernetes API 및 Gitops 워크 플로를 사용하여 데이터베이스, 스토리지 및 네트워킹과 같은 클라우드 리소스를 프로비저닝 및 운영하는 ACK 기능에 대한 개요를 제공합니다.그런 다음 선도적 인 기업이 ACK에 셀프 서비스 플랫폼을 구축하는 방법을 들어 개발자가 컨테이너화 된 응용 프로그램과 함께 AWS 인프라를 선언하고 배포 할 수 있도록 권한을 부여합니다.ACK를 사용하여 개발자 생산성을 높이고 클라우드 인프라의 생산 준비를 보장하기위한 모범 사례를 배우십시오.
