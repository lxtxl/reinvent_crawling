데이터 프라이버시를 유지하고 규정을 준수하기 위해 금융 기관은 데이터 처리 응용 프로그램의 공격 표면 영역을 적극적으로 줄여야합니다.이 Chalk Talk는 AWS Nitro Enclaves를 활용하여 개인 식별 정보 (PII)와 같은 민감한 데이터에 대한 보안을 향상시키는 방법을 설명합니다.암호화, 안전한 데이터 저장 및 지불 인증 프로세스 및 카드 소지자 정보 보호에 적용되는 고립 된 컴퓨팅 환경을 포함한 민감한 데이터를 안전하게 저장 및 처리하기위한 모범 사례를 살펴보십시오.Amazon API Gateway, AWS Lambda 및 Amazon DynamoDB를 포함한 AWS 서비스를 사용하여 데이터 보호 및 안전한 결제 처리의 복잡성을 탐색하는 방법을 배우십시오.
