    AWS services like Amazon DynamoDB and Amazon DocumentDB enable scalable applications using serverless, NoSQL database architectures. Amazon OpenSearch Service simplifies website search, log analytics, and application monitoring. With the zero-ETL integration between Amazon DynamoDB and Amazon DocumentDB with Amazon OpenSearch Service, customers can now seamlessly replicate data between databases and OpenSearch. This chalk talk explores strategies to harness the full power of OpenSearch to query transactional data in databases and enable advanced use cases like semantic and hybrid search using the vector engine in Amazon OpenSearch Service.
