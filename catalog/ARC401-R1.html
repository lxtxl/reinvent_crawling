    Join this hands-on workshop to explore a resilient, cloud-native architecture that surpasses the stringent availability and recovery regulations for financial markets utility providers. This highly available design uses Amazon ECS, Java Spring Boot, Amazon MQ, Amazon Kinesis, Amazon DynamoDB, Amazon Aurora, Amazon Application Recovery Controller, and AWS Systems Manager. Delve into key design decisions, including cross-Region data stores, messaging, and cross-Region state replication. Also learn about exactly-once transaction processing, reliable global traffic shifting, and best practices for fail-safe disaster recovery orchestration. You must bring your laptop to participate.
