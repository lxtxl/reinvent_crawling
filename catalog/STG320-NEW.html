    In this workshop, work with the new Amazon FSx Intelligent-Tiering elastic storage class. Learn to launch, configure, and optimize file systems that combine NAS capabilities with virtually unlimited scalability, elasticity, and automatic cost optimization. Through guided exercises, create Amazon FSx for OpenZFS file systems, implement automatic tiering, and explore features like snapshots, clones, and compression.
