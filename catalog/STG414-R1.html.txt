고객이 구조화되지 않은 엔터프라이즈 파일 데이터의 잠재력을 최대한 발휘하려고함에 따라 많은 사람들이 더 깊은 통찰력을 얻기 위해 생성 AI로 전환하고 있습니다.이 세션에서는 NetApp Ontap의 Amazon FSX를 사용하여 Amazon Bedrock의 지식 기반으로 회사의 온-프레미스 파일 데이터를 신속하고 안전하게 통합하는 방법을 알아보십시오.RAG (Resprieval-Augmented Generation) 워크 플로우를 구현하여 개인 데이터 소스의 상황 정보로 생성 AI를 풍부하게하여보다 관련성 있고 정확하며 맞춤형 응답을 보장하는 방법을 살펴보십시오.AI 중심의 통찰력을 최적화하기 위해이 데이터를 벡터 데이터베이스에 효율적으로 관리하고 저장하는 방법을 알아보십시오.
